# UX Audit Checklist

## Heuristic Evaluation (Nielsen's 10)

Check each item and note severity: Critical / Major / Minor / Cosmetic

- [ ] **Visibility of system status**: Does the user always know what's happening?
- [ ] **Match with real world**: Does the system speak the user's language?
- [ ] **User control and freedom**: Can users undo mistakes and navigate freely?
- [ ] **Consistency and standards**: Are patterns used consistently?
- [ ] **Error prevention**: Are errors prevented before they happen?
- [ ] **Recognition over recall**: Are options visible instead of requiring memorization?
- [ ] **Flexibility and efficiency**: Do expert users have shortcuts or accelerators?
- [ ] **Aesthetic and minimalist design**: Is every piece of information necessary?
- [ ] **Help users recognize, diagnose, and recover from errors**: Are error messages helpful?
- [ ] **Help and documentation**: Is help available when needed, without being intrusive?

## Task Flow Analysis

- [ ] Can a new user complete the primary task in under 30 seconds?
- [ ] Is the number of steps minimal (3-5 clicks/taps)?
- [ ] Are there any steps where users commonly get stuck?
- [ ] Is there a clear path back to the starting point?
- [ ] Does every screen clearly indicate where the user is and what to do next?

## Accessibility Scan

- [ ] Color contrast ≥ 4.5:1 for normal text
- [ ] All interactive elements keyboard-accessible
- [ ] Focus indicators visible on all interactive elements
- [ ] Screen reader can navigate the page logically
- [ ] Touch targets ≥ 44×44px

## Visual Design

- [ ] Visual hierarchy matches content importance
- [ ] Consistent spacing and alignment
- [ ] Typography scale is consistent
- [ ] Colors used meaningfully, not decoratively
- [ ] Empty, error, and loading states all handled
